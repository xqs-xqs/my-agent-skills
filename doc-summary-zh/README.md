# 📄 doc-summary-zh

**技术/课程文档结构化深度总结** — 一个专为中文用户设计的 Claude skill，把复杂的专业文档变成清晰、分层、可直接使用的结构化总结。

---

## ✨ 功能简介

上传一份课件、技术文档或学术讲义，这个 skill 会自动完成四层输出：

| 层级 | 内容 |
|------|------|
| 🔹 一句话总结 | 10 秒掌握全文核心 |
| 🔹 可视化思维导图 | 渲染为交互式知识框架图 |
| 🔹 分模块深度总结 | 概念 / 公式 / 流程 / 优劣逐一拆解 |
| 🔹 高频问题答疑 | 提炼 3-4 个核心考点与理解难点 |

所有输出严格贴合原文，不添加无依据的外部信息。

---

## 🎯 适用场景

- 课程课件 → 重点考点 + 核心算法 + 实操步骤
- 技术文档 → 底层原理 + 落地方法 + 适用边界与优劣对比
- 学术讲义 → 知识框架梳理 + 关键公式还原
- 行业教程 → 流程拆解 + 应用案例提炼

---

## 🚀 安装使用

1. 下载 [`doc-summary-zh.skill`](./doc-summary-zh.skill)
2. 在 Claude.ai 的 Settings → Skills 中上传该文件
3. 上传任意文档，说「帮我总结这份文档」即可触发

**触发关键词**：总结 / 概括 / 梳理 / 提炼知识点 / 帮我看看这个文档
